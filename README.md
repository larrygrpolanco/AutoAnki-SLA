# AutoAnki-SLA

Nation's framework distinguishes spoken form knowledge ('What does it sound like?') from written form ('What does it look like?'). Should we treat audio as its own card type (a 5th card: hear audio → recall meaning), or integrate audio INTO the existing 4 cards strategically?

For example sentences across the 4 card types — the research says diverse contexts build richer semantic networks. Should each card use a DIFFERENT example sentence for the same word (4 unique sentences = 4 encounters), or should some share sentences?

On the JSON schema: should the LLM generate ALL sentences upfront in one pass (more consistent but less adaptive), or should the philosophy doc specify sentence constraints so tightly that the LLM has clear guardrails?
